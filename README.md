# Navent interview challenge

Navent interview challenge developed in React.js. This project uses Node.js version 13.9.0.

**1.** Clone the repo:

```
$ git clone git@github.com:gicanas94/ejercicio-navent.git
$ cd ejercicio-navent
```

**2.** Install dependencies:

```
$ npm install
```

**3.** Run the app:

```
$ npm start
```
