# Navent interview challenge

Navent interview challenge developed in React.js. This app uses Node.js version 13.9.0.

**1.** Clone the repo:

```
$ git clone git@github.com:gicanas94/navent-challenge.git
$ cd navent-challenge
```

**2.** Install dependencies:

```
$ npm install
```

**3.** Run the app:

```
$ npm start
```
